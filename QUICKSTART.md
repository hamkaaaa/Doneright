# ⚡ Quick Start Guide

## Development

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev
```

Buka browser di `http://localhost:3000`

## Deploy ke Vercel

### Cara Tercepat:

1. Push ke GitHub:
```bash
git init
git add .
git commit -m "Initial commit"
git push
```

2. Buka https://vercel.com/new

3. Import repository Anda

4. Klik Deploy ✨

**Done!** Aplikasi live dalam 2 menit.
